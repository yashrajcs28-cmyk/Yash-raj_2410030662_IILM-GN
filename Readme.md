# Student Data Analysis Project

## 📌 Description
This project is a simple data analysis of student marks using Python.

## ⚙️ Features
- Reads student data from CSV file
- Calculates total and average marks
- Finds the topper of the class
- Saves updated data

## 🛠️ Technologies Used
- Python
- Pandas

## ▶️ How to Run
1. Install pandas:
   pip install pandas

2. Run the file:
   python analysis.py
